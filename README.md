# Результаты построения модели предсказания увольнения сотрудников

## Лучшая модель

Для прогнозирования увольнения сотрудников была выбрана модель:
- **Модель**: `DecisionTreeClassifier`
- **Параметры**: 
  - `criterion='log_loss'`
  - `max_depth=6`
  - `max_leaf_nodes=30`
  - `min_samples_leaf=3`
  - `min_samples_split=7`
  - `random_state=42`

## Качество модели

Модель удовлетворяет требованию ROC-AUC ≥ 0.91:
- **ROC-AUC на тестовой выборке**: 0.9234 ✅
- **ROC-AUC на обучающей выборке**: 0.9338

### Сравнение с базовой моделью
Модель значительно превосходит базовый классификатор:
- **DummyClassifier ROC-AUC**: 0.5017
- **Улучшение**: +0.4217

## Преимущества выбранной модели

1. **Тщательный подбор параметров** - перебрано больше гиперпараметров
2. **Устойчивость к выбросам** - меньше подвержена влиянию аномальных значений
3. **Хорошее обобщение** - близкие значения метрик на train и test выборках

## Значимые признаки

По степени влияния на прогноз (по убыванию):

1. **level** (`+`) - уровень занимаемой должности
2. **job_satisfaction_rate** (`-`) - уровень удовлетворенности сотрудника работой  
3. **salary** (`+`) - ежемесячная зарплата сотрудника
4. **workload** (`-`) - уровень загруженности сотрудника
5. **employment_years** (`+`) - длительность работы в компании

*Примечание: `+` - прямая зависимость, `-` - обратная зависимость*

## Заключение

Модель демонстрирует высокое качество прогнозирования и может быть использована для выявления сотрудников с риском увольнения. Наибольшее влияние на вероятность увольнения оказывают уровень должности и удовлетворенность работой.
